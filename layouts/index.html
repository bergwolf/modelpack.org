Quick Start

1

Install modctl

curl -sSL https://raw.githubusercontent.com/modelpack/modctl/main/install.sh | sh
2

Package Your Model

modctl build -t myregistry.com/mymodel:v1.0 /path/to/model
3

Push to Registry

modctl push myregistry.com/mymodel:v1.0
4

Use in Kubernetes

kubectl apply -f model-pod.yaml
